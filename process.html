Verification Process

The procedure is designed to ensure an accurate evaluation of each subject. Please follow all instructions.

01

Receive Mark

Subject receives the access mark and personal case ID before entering the station.

02

Enter Station

Subject enters the verification environment and remains seated until instructed otherwise.

03

Complete Assessment

Biometric scan, oath verification and four rounds of questioning are completed.

04

Archive Result

The final score is assigned to the case ID and marked as approved, limited or non-citizen.

05

Scan Code

Subject scans the hand mark and enters the case ID to access the result file.

06

View File

The result is displayed through the NVA portal. Result status is final.
